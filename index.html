Workshop on Introductory Python Programming for Mathematics Teachers

Bicol University invites you to attend the Workshop on Introductory Python Programming for Mathematics Teachers. This workshop is designed for high school and college math teachers, providing practical coding skills that can be directly applied in the classroom. It will help educators integrate Python programming into their math curriculum. The workshop will be held at Bicol University in Legazpi City, Albay, from February 27 to March 1, 2025. Join us for this 5-day event, which includes hands-on sessions where you will learn Python programming specifically for math education.
